# OKIZEME.GG Script
This is a script that simply filters moves by -15 and above since the filter does not exist in the website. 

## Download Link
https://github.com/haichteque/minus15filter-okizeme.GG-script/releases/download/v1.0/OkizemeFilter_v1.0.zip

## How to Install
2. Extract the zip
1. Go to Manage Extensions in your Browser
2. Enable Developer Mode
3. Click load unpacked
4. Select the OkizemeFilter folder
5. Enable the extension
